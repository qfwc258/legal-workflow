# 模板库使用指南

## 内置模板

系统内置以下模板分类：

### 民事类
- 民事起诉状
- 答辩状
- 上诉状
- 再审申请书

### 商事类
- 买卖合同
- 租赁合同
- 借款合同
- 股权转让协议

### 刑事类
- 刑事辩护词
- 刑事附带民事起诉状
- 取保候审申请书

### 执行类
- 执行申请书
- 财产保全申请书
- 限制高消费申请书

## 用户自定义模板

用户可将自定义模板放入 `templates/` 目录，系统优先使用用户模板。

## 模板变量

| 变量 | 说明 |
|------|------|
| {原告} | 原告姓名/名称 |
| {被告} | 被告姓名/名称 |
| {案由} | 案件类型 |
| {管辖法院} | 受理法院 |
| {诉讼请求} | 具体请求内容 |
