# Word文档默认格式规范（法律公文）

## 遵循标准
GB/T 9704-2012《国家行政机关公文格式》

## 页面设置

| 要素 | 设置值 |
|------|--------|
| 纸张大小 | A4 (21cm × 29.7cm) |
| 页边距-上 | 3.7cm |
| 页边距-下 | 3.5cm |
| 页边距-左 | 2.8cm |
| 页边距-右 | 2.6cm |

## 排版规范

| 要素 | 设置值 |
|------|--------|
| 正文行距 | 固定值 28磅 |
| 正文首行缩进 | 两中文字符（w:firstLineChars="200", w:firstLine="480"） |
| 正文字体 | 仿宋 / FangSong |
| 正文字号 | 三号 (14pt) |
| 条款标题字体 | 方正小标宋简体 |
| 条款标题字 | 三号 (14pt) 加粗 |
| 合同标题字体 | 方正小标宋简体 |
| 合同标题字号 | 二号 (22pt) 加粗 |
| 表格字体 | 仿宋 |
| 表格字号 | 小四号 (12pt) |

## XML首行缩进实现

```python
def set_first_line_indent_2chars(p):
    pPr = p._p.get_or_add_pPr()
    ind = OxmlElement('w:ind')
    ind.set(qn('w:firstLineChars'), '200')
    ind.set(qn('w:firstLine'), '480')
    pPr.append(ind)
```
