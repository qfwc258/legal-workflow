# Agent Memory - Word格式规范

## 记忆类型: 事实 (Fact)

**内容**:
Word文档默认格式遵循GB/T 9704-2012法律公文标准：
- 页面：A4 (21x29.7cm)
- 页边距：上3.7cm，下3.5cm，左2.8cm，右2.6cm
- 正文：仿宋，三号(14pt)，固定28磅行距
- 首行缩进：两汉字，XML设置 w:firstLineChars="200", w:firstLine="480"
- 条款标题：方正小标宋简体，三号(14pt)加粗
- 合同标题：方正小标宋简体，二号(22pt)加粗，居中
- 表格：仿宋，小四(12pt)
- 律师署名：陈伟律师，湖南金厚（宁乡）律师事务所，13975892485

**标签**: word, 格式规范, 公文, 法律

**来源**: 用户配置为生成Word文档的默认格式
